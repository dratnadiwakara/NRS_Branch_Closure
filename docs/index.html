    Deposit beta regressions, baseline closure and opening models, closure by time period, closures and branch usage (pandemic era), and reduced-form demographics.

    Two-stage bootstrap standard errors for branch closure regressions, accounting for generated regressor uncertainty.

    Baseline opening model, openings by time period, reduced-form demographics, and openings and branch usage (pandemic era).

    Summary statistics by bank size and cycle (Mid 2019, Early 2012), and univariate openings by deposit beta decile.

  • 7. Descriptive statistics – closure sample

    Bank-level characteristics, deposit beta regressions, beta distributions and correlations, univariate closure rates by decile, and branch-level descriptive statistics by size and cycle.
